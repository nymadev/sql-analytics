# sql-analytics
Project

Projeto prático de análise de dados utilizando PostgreSQL.

## Objetivos
- Aprender SQL na prática
- Trabalhar com modelagem relacional
- Criar queries analíticas
- Desenvolver portfólio para área de dados

## Tecnologias utilizadas
- PostgreSQL
- DBeaver
- Git/GitHub

## Estrutura do projeto
- Criação de tabelas
- Inserção de dados
- Índices
- Queries básicas, intermediárias e avançadas

## Conceitos praticados
- SELECT
- JOIN
- GROUP BY
- ORDER BY
- Window Functions
- CTEs
- Aggregations
- Índices

## Base do projeto
Sistema fictício de vendas contendo:
- clientes
- produtos
- vendas
